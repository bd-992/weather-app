# Weather App

A simple Python project that shows the current weather of a given city using an API.

### Features
- Fetches live weather data
- Displays temperature, humidity, and description
- Built with Python and requests library

# Weather App ☀️🌧️

Bu proje, Flask ve OpenWeatherMap API kullanarak şehirlerin anlık hava durumunu gösteren basit bir hava durumu uygulamasıdır.

## Özellikler:
- Şehir ismine göre hava durumu
- Sıcaklık, açıklama ve ikon bilgisi
- Türkçe dil desteği

## Kullanılan Teknolojiler:
- Python
- Flask
- Requests
- HTML/CSS (Jinja2 ile)
